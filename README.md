# ⛽ Kerala Fuel Cost Calculator

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://dannygeorge2026-coder-streamlit-fuel-app-app-45witn.streamlit.app/)

A smart Python Streamlit web app that calculates real-time monthly and trip-based fuel costs across Kerala by combining live fuel prices with OpenRouteService distance mapping.

### 🌐 Try the Live App!
**[Click here to use the Live Fuel Cost Calculator!](https://dannygeorge2026-coder-streamlit-fuel-app-app-45witn.streamlit.app/)**

---

### Features
* **Live Fuel Prices:** Scrapes real-time petrol and diesel prices specific to the user's district.
* **Smart Routing:** Uses OpenRouteService to calculate accurate driving distances and times between cities.
* **Monthly Estimates:** Calculates total monthly fuel expenditure based on average usage and vehicle mileage.
* **Statewide Analytics:** Automatically finds the districts with the highest and lowest fuel prices in Kerala.

### Built With
* Python
* Streamlit
* BeautifulSoup4
* OpenRouteService API
